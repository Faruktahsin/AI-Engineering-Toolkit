# Python Utilities

This directory contains practical Python scripts developed to automate repetitive tasks and improve productivity.

## Current Utilities

### 📂 File Organizer

Automatically organizes files into folders based on their file extensions.

Supported categories:

- Images
- Documents
- Videos
- Music
- Archives
- Spreadsheets
